Yet another static blog generator

This is my second attempt at starting a blog. Last time, well over 10 years ago, I think I used Blogger.com, or maybe it was a self hosted WordPress instance? Nowadays, static site generators seem to be the preferred option, but which one to choose? I played around a bit with popular choices Hugo and Pelican.

For documenting one of my other projects, I want to be able to run external programs to generate images and diagrams as part of building the site. This seems to be impossible with Hugo, where running external programs is deemed a security risk. Pelican has plugin support that would solve the problem, though. In the end, there was just too much flexibility for my tastes, and I decided to roll my own.

Using Python, geting something up and running was quick enough. It supports Markdown and uses Jinja2 templates. Syntax highligthing is also supported. Python example:

2
3
def main():
    print("Hello") #This line is highlighted
    print("world!)

Content is king, after all, so this will have to be good enough for now. For the curious, the source code can be found on GitHub.
